# Estacionamento e veículos
## Descrição do projeto
Um projeto que envolve ligação entre Backend e Frontend, a partir de um **Fork**, onde você pode cadastrar um veículo e realizar o tempo de estadia que seu carro ficará estacionado, assim, calculando o valor final de sua estadia após a retirada.

## Tecnologias utilizadas
- VS Code
- Live Server
- Navegador de sua preferência

## Passo a passo de como executar
1. Clone o repositório (git clone + url do repositório)
2. Abrir com o code
3. Abrir o **index** com o **Live Server**
4. Testar o Projeto
